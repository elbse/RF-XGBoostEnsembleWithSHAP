# Rubik - Android Malware Detector

## RF-XGBoost Ensemble with SHAP Explainability

A state-of-the-art Android malware detection system using ensemble machine learning (Random Forest + XGBoost) with SHAP values for model interpretability.

### Features

- **Threat Scanner**: Real-time APK file scanning
- **SHAP Explainability**: Feature contribution analysis
- **Model Analytics**: Performance metrics and evaluation
- **Dataset Insights**: Malware family distribution analysis
- **Research Comparison**: Benchmark against SOTA models
- **Feature Intelligence**: Permission risk assessment

# Clone the repository
